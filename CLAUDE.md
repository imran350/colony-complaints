# Colony Complaints System - Context

## Project Overview
- Built with: Node.js + Express + SQLite
- Frontend: HTML/CSS/JS (no frameworks)
- Location: D:\Work\claude\colony-complaints
- Database: complaints.db (SQLite file)

## How to Start
```bash
cd D:\Work\claude\colony-complaints
npm start
```
Then open http://localhost:3000

## Current Features
- Dashboard with stats
- Add/edit/delete complaints
- Filter complaints by date/trade/status
- Daily and monthly reports
- Export to CSV
- Print reports

## Trade Types
Plumber, Electrician, Carpenter, Painter, Sweeper, Mason, Gardener, Sewerage
